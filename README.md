# Todo-App

## Description
A basic todo app using react, react hooks and mobx-state-tree

To check live running visit [Todo-App](https://danielmejiadev.github.io/todo-app)

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start
```

## Test

```bash
$ npm run test
```
